Docker 学习笔记 - 命令


docker volume [COMMAND]

  • create 创建一个数据卷
  • inspect 显示一个或多个数据卷（volume）的详细信息
  • ls 列出所有 volume
  • prune 删除目前没人使用的数据卷 volume
  • rm 删除指定 volume
